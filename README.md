## Getting Started

Simply open `index.html` in your web browser to view the portal. No build process or dependencies required!

```bash
# Clone the repository
git clone https://github.com/eappell/Retirement-Portal.git

# Navigate to the directory
cd Retirement-Portal

# Open in your browser
open index.html  # macOS
start index.html  # Windows
xdg-open index.html  # Linux
```

## Adding New Tools

The portal is designed to easily accommodate new tools. To add a new tool:

1. Add a new tool card in the `tools-grid` section of `index.html`
2. Follow the existing card structure with icon, title, description, features, and button
3. Customize the SVG icon and colors to match your tool's theme
4. Link the button to your tool's page or application

## Design Philosophy

The portal emphasizes:
- **Visual Appeal**: Colorful gradients, engaging graphics, and smooth animations

- **Accessibility**: Semantic HTML, proper focus states, and smooth scrolling
